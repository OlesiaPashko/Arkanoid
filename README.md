# Arkanoid

This game was developed without Unity physics elements (colliders, rigibodies...).
There are
* full game loop
* ability to rotate, delete, move, add new obstacles in runtime
* specular reflection on move of ball
* Zenject for Dependecy Injection
* Unit tests
